# Progetto_TPS

Breve sito che funge da ricettario: permette di consultare ricette, cercare per ingredienti e ottenere una ricetta casuale. Realizzato con HTML, CSS, JavaScript e dati in formato JSON.

## Caratteristiche principali
- Visualizzazione di ricette (titolo, ingredienti, istruzioni, immagine).
- Ricerca per ingredienti: inserisci un ingrediente e mostra le ricette compatibili.
- Ricetta casuale: scegli automaticamente una ricetta a caso.
- Struttura dati in JSON.

## Tecnologie
- HTML (interfaccia) — [index.html](index.html)
- CSS (stile) — [style.css](style.css)
- JavaScript (logica: ricerca, filtro, random)
- JSON (archivio ricette)

## Struttura del progetto
- [index.html](index.html) — pagina principale.
- [style.css](style.css) — stili globali.
- ricette.json — file JSON con l'elenco delle ricette.
- utenti.json — file JSON con l'elenco degli utenti.
- immagini/ — cartella con le immagini delle ricette.


## Come usare
1. Apri [index.html](index.html) nel browser.
2. Usa il campo di ricerca per filtrare le ricette per ingredienti.
3. Clicca sul pulsante "Ricetta casuale" per visualizzare una ricetta scelta a caso.
