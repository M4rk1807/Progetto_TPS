# Progetto TPS

Progetto web per la gestione di ricette e utenti.

## Contenuto

- `index.html` - Pagina principale
- `ricette.html` - Visualizzazione ricette
- `aggiungi.html` - Aggiunta nuove ricette
- `form.html` - Modulo generico
- `registrazione.html` - Pagina di registrazione utenti
- `style.css`, `style1.css`, `style2.css` - Fogli di stile
- `ricette.json` - Database delle ricette
- `utenti.json` - Database degli utenti
- `Immagini/` - Cartella con le immagini del progetto

## Utilizzo

Aprire `index.html` nel browser per accedere all'applicazione.

## Struttura dati

### ricette.json
Contiene l'elenco delle ricette con ingredienti e istruzioni.

### utenti.json
Contiene i dati degli utenti registrati.
